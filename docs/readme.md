# Project Documentation

This folder contains project documentation and artifacts:

- `book_scraping_model.md` — data model and scraping details
- `db_models.md` — database schema & model descriptions
- `ddl.sql` — DDL SQL for reference
- `scraping_architecture.drawio` — architecture diagram (draw.io file)
- `uml/` — sequence and class diagrams (Mermaid + rendered SVG/PNG/HTML)

See `docs/uml/README.md` for instructions about rendering Mermaid diagrams and exporting images.
